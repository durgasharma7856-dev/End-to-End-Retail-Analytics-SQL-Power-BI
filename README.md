# End-to-End Retail Analytics 📊

## 📌 Project Overview
This project demonstrates an end-to-end retail analytics workflow—from SQL data engineering and database analysis to interactive business intelligence dashboards built in Power BI.

The objective of this project is to transform raw retail transaction data into meaningful business insights related to revenue growth, product performance, customer behavior, and inventory optimization.

---

## 🛠️ Tech Stack
- **Database:** PostgreSQL (SQL)
- **Visualization:** Power BI
- **Analytics:** DAX (Data Analysis Expressions)
- **Data Modeling:** Star Schema

---

## 🔍 Key Features

### SQL Data Engineering
- Designed a relational database with tables such as **CUSTOMERS**, **PRODUCTS**, and **SALES**
- Developed **25+ analytical SQL queries**
- Used **Joins, Aggregations, and Window Functions** for data preparation and analysis

### Interactive Dashboard
Created a **multi-page Power BI dashboard** including:

- **Overview:** Key KPIs such as Revenue, Profit, and Return Rate
- **Profit Analysis:** Identifying profit leakage and return reasons
- **Customer Insights:** Segmentation of high-value customers

### Business Logic
- Implemented tracking for **Dead Stock** (products with no sales for 90+ days)
- Built analysis for **profit optimization opportunities**

## 💡 Business Insights Derived

**Profit Leakage**  
Return reasons like **Wrong Item Delivered** significantly impact profitability.

**Category Performance**  
The **Electronics category** contributes the highest share of total revenue.

**Inventory Optimization**  
Dead stock items were identified to help optimize warehouse space and inventory management.
